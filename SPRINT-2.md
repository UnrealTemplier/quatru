# Текущие задачи (Спринт 2)

- Автоматический фокус клавиатуры при старте (без клика)

# Итог

**Проблема была не в коде, а в способе запуска.**

При запуске **из терминала** (Alacritty) — Alacritty очищает переменные `XDG_ACTIVATION_TOKEN` и `DESKTOP_STARTUP_ID` для дочерних процессов. Композитор Wayland (KDE Plasma) не может передать фокус окну без токена активации — фокус остаётся у терминала.

При запуске **из файлового менеджера/лаунчера** (Dolphin, KDE menu) — лаунчер передаёт токен активации, и окно автоматически получает фокус. Все клавиши (ESC, пробел) работают сразу.

**Решение:**
- `slint::set_xdg_app_id("quatru")` — устанавливает корректный app_id для идентификации композитором
- `forward-focus: key-handler` в `.slint` — маршрут начального фокуса
- `init => { self.focus(); }` в FocusScope — запрос фокуса при инициализации

**Запуск для тестирования:** из файлового менеджера или KDE меню, **не из терминала**.
