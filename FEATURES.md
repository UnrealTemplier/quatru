# Features — quatru

Постоянный список пользовательских фич.

## Реализовано (Спринты 1–3)

- Воспроизведение видео и аудио из файла (mpv через libmpv, OpenGL underlay)
- Пауза и продолжение воспроизведения (кнопка ▶/⏸ или пробел)
- Кастомная кнопка Play/Pause с анимациями hover/pressed (150ms)
- Кастомный TimelineSlider с анимированным thumb-ползунком (150ms)
- Мгновенный выход по нажатию Escape
- Закрытие окна по крестику
- Тёмный UI в стиле Spotify/DAW плагинов
- Кастомный контрол-бар внизу экрана (80px, opacity 0.95)
- Окно без системных декораций
- Запуск без файла — чёрный экран
- Запуск с файлом из командной строки
