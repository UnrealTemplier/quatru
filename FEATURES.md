# Features — quatru

Постоянный список пользовательских фич.

- Воспроизведение видео и аудио из файла
- Пауза и продолжение воспроизведения (кнопка ▶/⏸ или пробел)
- Прогресс-бар с отображением текущей позиции (пока не отображает и нельзя делать seek по нему)
- Таймкод в формате «прошло / всего» (пока только заглушка, не отображает реального времени)
- Мгновенный выход по нажатию Escape
- Закрытие окна по крестику
- Тёмный UI в стиле Spotify/DAW плагинов
- Кастомный контрол-бар внизу экрана
- Окно без системных декораций
- Запуск без файла — чёрный экран
- Запуск с файлом из командной строки
